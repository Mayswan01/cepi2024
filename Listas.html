Trabalhando com Listas

Listas Ordenadas

Listas ordenadas são listas que servem para organizar itens que possuem sequencial lógico. A lista possue a TAG OL e os itens tem RAG LI

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item

Listas não ordenadas

Listas não ordenadas são listas que não possuem uma sequencia lógica, geralmente ultilizar marcadores e usa-se a TAG UL para a lista e LI para o item.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item
